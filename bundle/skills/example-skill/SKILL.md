# 範例 Skill — 給 Codex 看格式

## 什麼時候用這個 skill

當 codex 要建你的第一個 skill、需要一個格式參考時，會 read 這個檔案。

## 核心心法

1. **frontmatter 的 description 是市集招牌** — 寫得太抽象別人不會買
2. **內容要有具體故事** — 沒故事的 skill 跟維基百科沒兩樣
3. **限縮場景** — 一個 skill 只解決一個情境，不要塞太多
